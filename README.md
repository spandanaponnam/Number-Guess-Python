# Python CLI Games 🎮

A collection of simple Python command-line games to practice programming and have fun!  

This repo includes:

1. **Number Guessing Game** – Guess the number chosen by the computer.  
2. **Snake-Water-Gun Game** – A simple CLI game similar to Rock-Paper-Scissors.

**Features**

Simple command-line interface, easy to run.

Interactive and fun games for practice.

Perfect for beginners learning Python logic and control structures.

**Installation:**

Clone the repository:

git clone https://github.com/<your-username>/python-cli-games.git
cd python-cli-games

Make sure Python is installed (Python 3.x recommended).

**Usage**

Run any game by executing the corresponding Python file:

python number_guessing.py
python snake_water_gun.py
